# ExternalESP-Shizuku

No-Root External ESP for 64-bit IL2CPP Android games using Shizuku.

## Features
- 100% External (separate app)
- Shizuku for memory reading
- Transparent Overlay Canvas ESP

## Setup
1. Install Shizuku + activate via Wireless Debugging
2. Grant Shizuku permission to this app
3. Run the game + this ESP app

**Warning**: For educational purposes only. Game bans possible.
